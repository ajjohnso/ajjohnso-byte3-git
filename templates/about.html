Accountability: About This Data Set

This dataset contains my activity & phone location data between 1/30/17 and 2/22/17. There are two relevant tables for this Byte:

  • locations_visit: The table that contains locations I have visited. It includes columns that represent the timestamp, address based on my location, and reported accuracy of each measurement.
  • locations:The table that contains location data. It includes columns that represent the timestamp, geographical coordinates, accuracy, bearing, and speed of each measurement.

    • Where did this data come from?

    This data was collected using AWARE, which is an Android framework dedicated to instrument, infer, log and share mobile context information, for application developers, researchers and smartphone users. AWARE captures hardware-, software-, and human-based data. The data is then analyzed using AWARE plugins, to help transform data into information you can understand.

    Who has access to it?

    Data collected for this Byte is not publically accessbile, however, if you are interested in collecting your own data using this framework you can do so here.
